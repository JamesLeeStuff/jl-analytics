Pitch Code Format
[Pitch][Loc][Result][V1][V2]
e.g. 11S90 = Slot 1 · Upper Inside · Strike · 90mph
Velo accepts decimals: 90.1 or 901 both log as 90.1

CODE KEY

Each pitch is logged as a 5-character code. Here's how to build one.

1
Pitch Type
+
1
Location
+
S
Result
+
90
Velocity
Example
11S90
Fastball · Upper Inside · Strike · 90 mph
Digit 1 — Pitch Type
1Fastball (FB)
2Curveball (CB)
3Slider (SL)
4Changeup (CH)
Digit 2 — Location (RHH view)
1Upper Inside
2Lower Inside
3Upper Outside
4Lower Outside
0Middle Middle
Height Guideline
Upper zones (1, 3): Belt and above
Lower zones (2, 4): Below the belt
For left-handed hitters, inside/outside flips automatically.
Digit 3 — Result
SStrike (in zone)
BBall
CChase (swing out of zone)
Special: use 0B (digit 2 = 0, digit 3 = B) for a middle-middle pitch the ump missed.
Digits 4–5 — Velocity (mph)
9090 mph
8585 mph
7878 mph
Always two digits. Use the last two digits of the speed — e.g. 92 mph = 92.
Quick Reference Examples

13B85 Fastball · Upper Outside · Ball · 85 mph
24S78 Curveball · Lower Outside · Strike · 78 mph
32C83 Slider · Lower Inside · Chase · 83 mph
10B88 Fastball · Middle Middle · Ball (ump miss) · 88 mph
41S76 Changeup · Upper Inside · Strike · 76 mph
